Skill set

HTML
HTML (HyperText Markup Language) is the foundation of web development. It structures content on the web using elements like headings, paragraphs, images, and links.
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of web pages. It works alongside HTML to style elements like colors, fonts, spacing, and animations.
Java
Java is a powerful, object-oriented programming language used for building web applications, mobile apps (Android), desktop software, and enterprise systems. It is known for its platform independence, meaning code written in Java can run on different operating systems using the Java Virtual Machine (JVM).
Node.js
Node.js is a powerful, open-source runtime environment that allows developers to run JavaScript on the server-side. It is built on Google Chrome's V8 engine, making it fast and efficient for building scalable applications.
React
React.js is a popular JavaScript library for building dynamic and interactive user interfaces, especially for single-page applications (SPAs). Developed by Facebook (Meta), React makes it easy to create reusable UI components.
